# Python Backend for Political Transparency Project

This backend uses the **Congress.gov API** (Library of Congress) - no API key required!

## Requirements

- Python 3.11+
- Dependencies: fastapi, uvicorn[standard], httpx, python-dotenv, pydantic

## Quick Start

```bash
cd backend
pip install -r requirements.txt
python main.py
```

Visit http://localhost:8000/docs for API documentation

## Data Source

- **Congress.gov API** - Official API from the Library of Congress
- Free and open to use, no API key needed
- Provides current congressional member data for the 118th Congress
